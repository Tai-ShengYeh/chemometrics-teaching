## 引擎 1｜概念定位稿

| 項目 | 內容 |
|------|------|
| **總概念** | PLS 同時抓住光譜的變異「與」它跟目標的相關性，建立可信的定量預測模型 |
| **子概念 1** | 監督式校正：用「光譜＋標準答案」訓練，新樣本只量光譜就能預測 |
| **子概念 2** | PLS vs PCA：PCA 只看 X；PLS 用 y 引導，潛在變量最大化 cov(X, y) |
| **子概念 3** | 用交叉驗證選潛在變量數（避免過度擬合），用獨立測試集驗證（R²、RMSEP、RPD） |
| **常見誤解** | 「潛在變量越多越準」→ 錯，超過轉折點只是在背雜訊、過度擬合 |
| **帶走一句話** | PLS：一手抓光譜、一手抓答案，把幾秒的掃描變成可信的數字 |
| **最小事實包** | 監督式、校正 / 驗證、潛在變量 LV、RMSECV、R²、RMSEP、RPD、過度擬合 |

## 逐頁腳本

### P1｜封面（0–15s，T0 黑底）
* **字卡**：PLS 回歸／從近紅外光譜「預測」脂肪含量
* **字級**：標題 128px / 副標 52px
* **旁白**：上一集，PCA 讓我們看見光譜裡的結構。但如果我們真正想要的，是一個能直接預測脂肪含量的模型呢？這一集，主角是偏最小二乘回歸，PLS。
* **字幕**：PLS：把光譜變成可預測的脂肪數字

### P2｜為什麼要預測（15–32s，T2 左右）
* **視覺**：左「濕化學：慢、耗藥、破壞樣本」；右「NIR：幾秒、不破壞」
* **字卡**：兩種測脂肪的方法
* **旁白**：傳統測脂肪，要靠濕化學分析，費時、耗試劑、還會破壞樣本。而近紅外光譜，只要幾秒、完全不破壞樣本。我們的目標很明確：用光譜 X，去預測脂肪含量 y。
* **字幕**：目標：用光譜 X 預測脂肪 y，又快又不破壞

### P3｜監督式校正（32–51s，T3 SVG）
* **視覺**：流程圖 — 一批樣本（光譜 X + 真值 y）→ 訓練 → 模型 →（新樣本只量光譜）→ 預測
* **字卡**：先「校正」，再上線
* **旁白**：這是一個監督式問題。我們先準備一批樣本，同時有光譜，也有用標準方法測到的脂肪真值。用它們訓練出一條校正模型；之後新的樣本，只要量光譜，就能算出脂肪。
* **字幕**：校正：用「光譜＋真值」訓練出預測模型

### P4｜PCA vs PLS（51–72s，T3 SVG）★ 核心
* **視覺**：同一團點雲，teal 箭頭沿 X 最大分散方向，coral 箭頭「轉向」貼近 y 的梯度方向
* **字卡**：PCA 看 X，PLS 一手 X、一手 y
* **旁白**：為什麼不直接用 PCA？因為 PCA 只追求抓住光譜本身的變異，那個方向，不一定跟脂肪有關。PLS 不一樣，它同時考慮光譜的變異，和它跟脂肪的相關性，找出真正能拿來預測的方向。
* **字幕**：PLS 找「同時解釋光譜又預測脂肪」的方向

### P5｜潛在變量（72–90s，T4 公式）
* **視覺**：KaTeX `\max\ \operatorname{cov}(Xw,\ y)`
* **字卡**：潛在變量 Latent Variable
* **旁白**：這些方向，叫潛在變量 LV。PLS 讓每一個潛在變量，最大化光譜 X 和脂肪 y 之間的共變異。一句話：PCA 看 X 自己，PLS 一手抓 X、一手抓 y。
* **字幕**：潛在變量：最大化 X 與 y 的共變異

### P6｜前處理與切分（90–109s，T2 左右）
* **視覺**：左 SNV+一階導數示意；右 240→180/60 切分動畫
* **字卡**：前處理 + 訓練 / 測試切分
* **旁白**：實作上，我們先對光譜做 SNV 和一階導數，凸顯吸收峰。再把 240 個樣本切成兩份：180 個拿來訓練校正，60 個完全留到最後測試，模擬沒看過的新樣本。
* **字幕**：180 訓練 / 60 測試，留一份驗證真實表現

### P7｜該用幾個 LV？（109–131s，T1 大圖）★
* **視覺**：fig06_pls_rmsecv.png（曲線繪出，elbow 第 4 點脈動，灰區標 overfit）
* **字卡**：交叉驗證選轉折點
* **旁白**：潛在變量要用幾個？用交叉驗證。隨著潛在變量增加，誤差先快速下降，到第四個之後就幾乎不動了。再加上去，只會讓模型背下雜訊、過度擬合。所以我們選在轉折點：四個潛在變量。
* **字幕**：交叉驗證選在轉折點：4 個潛在變量

### P8｜成果：預測 vs 真實（131–152s，T1 大圖）★ 核心
* **視覺**：fig07_pls_pred_vs_meas.png（散點淡入 → 1:1 線繪出 → R² 爆出）
* **字卡**：預測值 vs 真實值
* **旁白**：成果如何？把預測值對真實值畫出來，校正集和測試集的點，都緊貼在一比一的對角線上。測試集的決定係數 R²，高達 0.96。模型從沒看過這 60 個樣本，依然預測得很準。
* **字幕**：測試集 R²=0.96，點緊貼 1:1 對角線

### P9｜模型有多好？（152–170s，T5 卡片）
* **視覺**：三張卡片
  - 卡片 1：**R² = 0.96** — 解釋了 96% 的變異
  - 卡片 2：**RMSEP ≈ 2.8%** — 平均預測誤差
  - 卡片 3：**RPD = 5.2** — >3 堪用，>5 優秀
* **字卡**：三個判讀指標
* **旁白**：怎麼判斷夠不夠好？RMSEP 是預測誤差，這裡大約 2.8 個百分點；RPD，是誤差相對於樣本變異的倍數，達到 5.2。一般 RPD 大於 3 就算堪用，超過 5，屬於優秀。
* **字幕**：RMSEP≈2.8%、RPD=5.2（>3 堪用，>5 優秀）

### P10｜回歸係數看化學（170–188s，T2 左圖右文）
* **視覺**：fig08_pls_coef.png，928 nm 尖峰高亮
* **字卡**：模型在「看」哪裡？
* **旁白**：再看模型的回歸係數。最大的尖峰，落在 928 奈米，正是脂肪的 C–H 吸收帶。這代表模型不是亂猜，而是真的盯著脂肪的化學訊號在預測。可解釋，才可信。
* **字幕**：係數尖峰在 928 nm = 脂肪 C–H 訊號

### P11｜應用與限制（188–206s，T5 卡片）
* **視覺**：兩張卡片（✓ 應用 / ⚠ 限制）
* **字卡**：能用在哪？要小心什麼？
* **旁白**：這樣的模型，可以裝在生產線上做即時品管，取代費時的實驗室分析。但要注意：校正模型只在它見過的範圍內可靠，遇到新品種、新製程，必須重新驗證，不能直接外推。
* **字幕**：可上線即時品管，但超出範圍要重新驗證

### P12｜PCA ↔ PLS 知識地圖（206–227s，T3 SVG）★
* **視覺**：分岔地圖 — 光譜→前處理 →（探索：PCA→分數/負荷量）/（預測：PLS→選 LV→測試集驗證）
* **字卡**：兩集合一張地圖
* **旁白**：把兩集連起來：光譜先前處理；想探索結構、找異常，用非監督的 PCA；想定量預測，就用監督的 PLS，再用交叉驗證選維度、用獨立測試集驗證。這就是近紅外做食品分析的核心地圖。
* **字幕**：探索用 PCA，預測用 PLS，獨立測試驗證
