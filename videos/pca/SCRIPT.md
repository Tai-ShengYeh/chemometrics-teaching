## 逐頁腳本

### P1｜封面（0–15s，T0 黑底）
* **字卡**：主成分分析 PCA／看見近紅外光譜裡的隱藏結構
* **字級**：標題 128px / 副標 52px
* **旁白**：近紅外光譜，能在不破壞食品的情況下，幾秒內看穿成分。但一條光譜有上百個數字，我們要怎麼讀懂它？這一集，我們從主成分分析 PCA 開始。
* **字幕**：一條光譜上百個數字，PCA 幫我們讀懂

### P2｜100 維的難題（15–37s，T1 大圖）
* **視覺**：fig01_raw_spectra.png（240 條依脂肪上色的原始光譜）
* **字卡**：240 個樣本 × 100 個波長
* **旁白**：這是 Tecator 資料集：240 個肉品樣本，每一個都在 850 到 1050 奈米之間，量了 100 個波長的吸光值。畫出來就像這樣，上百條交纏的曲線。每個樣本，都是 100 維空間裡的一個點，人眼根本無法直接想像。
* **字幕**：每個樣本是 100 維空間的一點，看不出來

### P3｜SNV 前處理（37–55s，T2 左圖右文）
* **視覺**：左 fig01 → fig02 交叉淡化；右文字
* **字卡**：先校正散射：SNV
* **旁白**：分析前，光譜通常要先前處理。樣本顆粒大小、量測距離的差異，會讓整條基線上下飄移。標準常態變量轉換，SNV，把每一條光譜各自校正，讓真正的化學差異浮現出來。
* **字幕**：SNV 校正散射，凸顯真正的化學差異

### P4｜PCA 的直覺（55–73s，T3 SVG）★
* **視覺**：散布點雲 → PC1 沿最大分散方向畫出 → PC2 垂直彈出
* **字卡**：找出資料最分散的方向
* **旁白**：主成分分析的想法很直覺。把所有樣本想成一團散開的點，PCA 要找出這團點最分散的方向，當作第一主成分 PC1；再找一個跟它垂直、第二分散的方向，當作 PC2。新的座標軸，沿著資訊最多的方向。
* **字幕**：PC1＝最大分散方向，PC2 與它垂直

### P5｜分數與負荷量（73–93s，T4 公式）
* **視覺**：KaTeX `X \approx T P^{\top}`，T、P 分別高亮
* **字卡**：分數 T、負荷量 P
* **旁白**：用數學講，PCA 把光譜矩陣 X，拆成分數 T 乘上負荷量 P 的轉置。分數，是每個樣本在新座標軸上的位置；負荷量，告訴我們每個波長怎麼組成這些新方向。一個看樣本，一個看變數。
* **字幕**：X ≈ T·Pᵀ：分數看樣本，負荷量看波長

### P6｜要留幾個成分？（93–115s，T1 大圖）★
* **視覺**：fig03_pca_scree.png（bars 升起 + coral 累積線繪製）
* **字卡**：陡坡圖：3 個成分就夠
* **旁白**：那要保留幾個主成分？看陡坡圖。光是 PC1 就抓住了將近 69% 的變異；加上 PC2 來到 97%；前三個主成分，更高達 99%。原本 100 個波長，幾乎可以濃縮成 3 個數字，資訊卻幾乎沒有損失。
* **字幕**：前 3 個主成分捕捉 99% 的變異

### P7｜分數圖：自動排序（115–135s，T1 大圖）★ 核心
* **視覺**：fig04_pca_scores.png（點由灰轉 teal→coral 依脂肪染色）
* **字卡**：每個點＝一個肉品樣本
* **旁白**：把每個樣本畫在 PC1、PC2 上，就是分數圖。每一個點，是一個肉品樣本，顏色代表脂肪含量。神奇的是，PCA 從頭到尾都沒看過脂肪數值，樣本卻沿著 PC1，由脂肪高低排得整整齊齊。結構，自己浮現了。
* **字幕**：PCA 沒看過脂肪，樣本卻依脂肪自動排列

### P8｜找出異常樣本（135–152s，T1 大圖）
* **視覺**：fig04 的 Hotelling T² 橢圓高亮，圈外點閃示
* **字卡**：分數圖也是品管利器
* **旁白**：分數圖也是品管的利器。這個虛線橢圓，是 95% 的信賴範圍；落在圈外的樣本，代表它和大多數樣本明顯不同，可能是異常、或量測出錯，值得拉出來檢查。
* **字幕**：落在 95% 橢圓外 = 可疑樣本，要檢查

### P9｜負荷量回連化學（152–171s，T2 左圖右文）
* **視覺**：fig05_pca_loadings.png，930 nm 帶高亮
* **字卡**：負荷量說的是化學
* **旁白**：負荷量則把我們帶回化學。PC1 和 PC2 的負荷量，在 930 奈米附近都有明顯起伏，那正是脂肪 C–H 鍵的吸收帶。PCA 沒有任何化學知識，卻自己重新發現了脂肪的訊號。
* **字幕**：930 nm 起伏 = 脂肪 C–H 吸收帶

### P10｜PCA 是 / 不是（171–187s，T5 卡片）
* **視覺**：三張卡片逐塊淡入
  - 卡片 1：**非監督** — 不需要標準答案
  - 卡片 2：**擅長** — 探索、分群、找異常
  - 卡片 3：**不做** — 不直接預測含量
* **字卡**：認清 PCA 的定位
* **旁白**：要記得，PCA 是非監督式的：它只看光譜本身，不需要任何標準答案。它擅長探索資料、分群、找異常；但它不會直接告訴你脂肪是幾趴。
* **字幕**：PCA 非監督：探索分群找異常，不做預測

### P11｜知識地圖（187–206s，T3 SVG）★
* **視覺**：中心輻射 — 光譜 → 前處理 → PCA →（分數 / 負荷量 / 陡坡）三分支
* **字卡**：PCA 探索流程全圖
* **旁白**：把流程串起來：原始光譜，先做 SNV 前處理，再用 PCA 拆成分數、負荷量和陡坡圖。分數看分群、負荷量看化學、陡坡圖決定維度。這就是用 PCA 探索食品光譜的完整地圖。
* **字幕**：光譜 → 前處理 → PCA → 分數 / 負荷量 / 陡坡
